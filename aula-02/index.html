HTML e CSS

Tag é um código usado para marcar o início e, onde for requerido, o fim de um elemento HTML. Há, como exposto acima, tags de abertura e de fechamento. Uma tag de abertura é representada por sinal de menor ( < ), um nome de elemento HTML, e um sinal de maior ( > ) e deve ser colocada imediatamente antes do início do conteúdo do elemento. Uma tag de fechamento se diferencia de uma tag de abertura apenas por uma barra ( / ) antes do nome do elemento e deve ser colocada imediatamente após o fim do conteúdo do elemento.
